active infinitive with passive meaning

Posted on 2026-04-10 In Grammar

“to do ~” という書き方で、能動態の形をしているが受動態の意味をとる例がある。

■ 形容詞 + to do

不定詞の意味上の主語が、文の主語と一致しないとき、能動形でも受動の意味になる
※Tough構文という考え方もある ( Tough movement )

  • This book is easy to read.
    この本は読まれやすい（読みやすい）
    → book は「読む側」ではなく「読まれる側」なので、to read の意味は受動。

  • The problem is difficult to solve.
    → その問題は解かれにくい

  • The water is safe to drink.
    → その水は飲んでも安全（＝飲まれても安全）

  • よく使われる形容詞

    • easy / difficult / hard
    • safe / dangerous
    • pleasant / fun
    • impossible / possible
    • ready / tough / convenient

■ 名詞 ＋ to do

名詞を後ろから修飾するとき、その名詞が不定詞の目的語の関係にある場合、能動態のままで「〜されるべき」という意味を含む。

  • I have a lot of work to do.
    (直訳：やるべき仕事がたくさんある。)
    (解釈：仕事の側から見れば「されるべき」仕事。)

  • There is no time to lose.
    → 無駄にできる［失われるべき］時間はない

  • There is nothing to see.
    → 見られるものは何もない

■ to 不定詞以外で、「形は能動なのに意味は受動になる」例

自動詞の能動形なのに受動の意味をもつ例、中間態（middle voice）

特定の動詞が、受動態の形（be + 過去分詞）を使わずに、「〜される」という意味で使われるケース
能動形なのに、意味は受動的。
「性質・特徴」を述べるときに使われる。

  • This book sells well.
    → この本はよく売れる（＝よく“売られている”）

  • The door opens easily.
    → このドアは簡単に開く（＝開けられる）

  • The car drives smoothly.
    → この車はスムーズに運転できる（＝運転されるとスムーズ）

  • This cake cuts easily.
    → このケーキは簡単に切れる

  • The sign reads “Keep Out.”
    → 看板には「立入禁止」と書いてある［読める］

need / want / require + -ing（意味上の受動）

動名詞 -ing が能動形なのに、意味は受動。
動詞 need, want, deserve, require などの後ろに動名詞（-ing）が来るパターン

  • The car needs washing.
    → 車は洗われる必要がある（＝洗う必要がある）
    (※ needs to be washed と書き換え可能)
    アメリカ英語では needs to be washed が一般的、
    イギリス英語ではこの「能動形で受動の意味」がよく使われる

  • Your hair wants cutting.（英）
    → 髪は切られる必要がある

  • The machine requires fixing.
    → 修理される必要がある

  • The house wants painting.
    → その家はペンキを塗る必要がある

  • This problem deserves thinking about.
    → この問題は検討に値する

形容詞＋doing（意味上の受動）

worth の後の動名詞は能動形でも受動の意味。

  • This movie is worth watching.
    → 見る価値がある（＝見られる価値がある）
    ※ worth being watched と言わない

  • The idea is worth considering.
    → 考えられる価値がある

形容詞＋前置詞＋doing（意味上の受動）

特に be ready for doing など。

  • The report is ready for printing.
    → 印刷される準備ができている

  • The food is good for eating.
    → 食べるのに適している（＝食べられるのに適している）

前置詞 ＋ 名詞

前置詞と名詞が組み合わさって、「今まさに〜されている最中だ」という受動的な進行状態を表すことがある

  • The road is under construction.
    → 道路は建設中だ。 (建設されている)

  • The movie is now in production.
    → その映画は現在制作中だ。 (制作されている)

  • The plan is under consideration.
    → その計画は検討中だ。 (検討されている)
